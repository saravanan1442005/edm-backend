How to Provision

QR Code Method (Company Owned)

  1. Factory reset the Android device.
  2. Tap the welcome screen exactly 6 times in the same spot.
  3. Connect to Wi-Fi if prompted.
  4. Scan the generated QR code above.
  5. The device will automatically download the agent and enroll.

Token Method (BYOD / App Installation)

  1. Install the EDM companion application directly on the device.
  2. Open the app and select "Enroll Device".
  3. Enter or paste the token text generated by clicking "Generate Token".
